- ZFC: 策梅洛-弗兰克尔集合论
    核心思想：所有数学对象（数、函数、空间等）都可以用 集合 这个概念构造和描述

- 集合列：按某种顺序排列的一列集合

- 关系：笛卡尔积的子集
    - 等价关系: 自反性、对称性、传递性
        相等、同余、同一类别
        等价类两两不相交
    - 序关系
        偏序关系：自反性、反对称性、传递性

- 环 Ring 是一种代数结构，用于抽象加法和乘法同时存在的体系
    有限并封闭
    有限交封闭
    不一定包含全集X

- sigma ring
    是一个集合环且额外满足：对可数并封闭

- sigma 代数 对补集封闭

- 集合的大小
    - 外侧度
        1. 零空集
        2. 单调性
        3. 可数可加性（次可加性）
    - 测度

- 连续统
    基数等于实数集 \mathbb R 的集合
    像实数那样连续的整体
